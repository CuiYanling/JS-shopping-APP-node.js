# nodejs-mysql-mall
基于nodejs + mySQL + express的商城后台

## 安装api文档插件
```
npm i apidoc -g
```

## 进入项目
```
cd {项目目录}
```

## 安装依赖包
```
$ cnpm i
```

## 还原数据库
1. 在mysql中创建mall数据库
2. 将mall.sql文件还原至mall数据库

## 重新生成API文档
```
$ npm run api
```

## 启动
```
$ npm start
```

## 后台API文档地址
```
http://localhost:3000/api/
```

## 商城管理后台地址
```
http://localhost:3002/admin/
```
